# 《故事创意指南》结构化笔记

## 1. 这本书的核心目标

这本书不是单纯讲“文笔”，而是讲**如何从零构思一个能写下去、有人愿意读下去的小说/剧本故事**。作者反复强调：小说/剧本本质上是讲故事，初学者最先要解决的不是主题、立意、文采，而是把故事讲得“让读者不看到最后不撒手”。

它主要适合两类人：一是零基础小说/剧本写作者，二是基础薄弱、经常卡文、故事写到几万字就推倒重来的人。作者认为卡文的根本原因通常是：对故事构思不够细，对人物了解不够深。

---

## 2. 总体写作流程

这本书可以归纳成一条完整流程：

**心理准备 → 故事构思 → 难题设计 → 人物小传 → 人物展示 → 对话设计 → 悬念设计 → 切入点/开头**

作者在导读里把全书分成七个重点：心理素养、故事构思法、跌宕起伏的故事设计、人物塑造、对话、悬念、切入点。

最重要的一句话是：

> 故事不是先写出来的，而是先构思出来的。

构思越细，写作时越不容易卡；人物越清楚，人物出场、行动和台词就越自然。

---

## 3. 写作者的底层心态

作者反对“天赋决定论”。他认为写作是一门技术活，重点是学习、练习和长期坚持，而不是一开始就判断自己有没有天赋。书中把写作类比为木匠做桌子：懂原理不等于会做，必须反复实操。

这里可以提炼成三条写作心态：

| 心态    | 含义                 |
| ----- | ------------------ |
| 不迷信天赋 | 不要用“我没天赋”当作放弃理由    |
| 接受慢回报 | 写作前期可能长期没有收益       |
| 先认同身份 | 先把自己当作作家，再按作家的标准行动 |

作者尤其强调“作家的自我身份认同”：不是作品让你成为作家，而是你先以作家的方式生活、观察、构思、写作，作品才会出现。

---

## 4. 故事构思的核心方法

书里最实用的部分是两个构思法：

# A. 问题构思法

用五个问题搭出故事框架：

| 问题      | 作用    | 你要写下的内容    |
| ------- | ----- | ---------- |
| 他是谁？    | 确定主人公 | 身份 + 缺陷/挫折 |
| 他要做什么？  | 确定任务  | 主人公必须完成的目标 |
| 他为什么做？  | 确定动机  | 不做不行的理由    |
| 他准备怎么做？ | 确定计划  | 大致行动路线     |
| 结果是什么？  | 确定结局  | 成功、失败或代价   |

作者认为主人公不能只是一个普通身份，最好一开始就带着“生活缺陷”或“变故”。比如：被追杀的公主、被丈夫抛弃的富太太、幸存者首领。没有变故，就没有故事。

# B. 难题构思法

难题构思法用于细化情节。它的核心是：

**问题 + 解决问题的方法 = 故事情节**

具体步骤：

1. 用一句话概括故事
   公式：**人物 + 任务 + 结局**
   例如：“赤焰军幸存者的复仇权谋之路。”

2. 找到故事结局
   结局是作者的目标，也是人物行动的方向。

3. 找出正派和反派
   反派不一定是人，也可能是灾难、制度、怪物、疾病、主人公自己的缺陷。

4. 给主角设计几个大难题
   难题要层层升级，不能平均用力。作者特别强调：故事不是先构思人物，而是先构思计划；有了计划，执行计划和阻碍计划的人物自然就会出现。

## 5. 好故事的关键：难题升级

书中用《白雪公主和七个小矮人》说明：成功故事里，主人公的难题不是孤立的，而是**一个比一个大，并且环环相扣**。例如白雪公主先失去母亲，再被继母追杀，再无家可归，再被毒死，最后继母变成女巫来杀她。

可以提炼成一个设计原则：

**每一个难题都要比上一个更严重；每一个难题都要由上一个难题引发。**

错误写法：

> 主人公遇到 A 麻烦，又突然遇到 B 麻烦，然后又突然遇到 C 麻烦。

正确写法：

> A 麻烦导致 B，B 的解决方式引出 C，C 逼出最终决战。

## 6. 三段式故事结构

书中把故事大致分成三段：

| 段落  | 功能 | 核心问题           |
| --- | -- | -------------- |
| 第一段 | 起因 | 主人公为什么必须行动？    |
| 第二段 | 对抗 | 主人公如何一次次闯关？    |
| 第三段 | 决战 | 主人公和敌人的恩怨如何了结？ |

其中第二段最重要。作者认为第二段就是“主人公和敌人多次生死较量的过程”，作者要借反派之手不断阻止主人公达成目标。

第二段的构建公式：

**找到敌人 → 设计关卡 → 设计过关方法 → 每关至少一波三折**

“一波三折”的意思是：一个关卡里，人物不能轻松过关，而要经历至少几次反转。

例子结构：

> 出城失败 → 化妆成功蒙混过关 → 被识破抓进大牢 → 发现暗道逃脱

这样读者才会有“糟糕—好转—更糟—逆转”的阅读体验。

## 7. 人物塑造方法

作者认为人物小传有三个作用：

1. 帮助作者真正了解人物
2. 塑造出性格鲜明的人物
3. 反过来检查故事大纲的漏洞

人物小传不是人物档案那么简单，而是人物形象设计过程。作者强调，如果作者不了解人物，就容易把不符合人物性格的台词和行动硬塞给他，导致故事崩塌。

人物小传至少包括：

| 模块   | 内容                  |
| ---- | ------------------- |
| 核心性格 | 这个人最底层的性格驱动力        |
| 生理属性 | 性别、年龄、外貌、身体状况、穿着、姿态 |
| 自然属性 | 家庭、教育、阶层、地域、经济条件    |
| 成长经历 | 重大创伤、重要事件、人生转折      |
| 人际评价 | 别人怎么看他              |
| 讨喜点  | 为什么读者愿意在乎他          |
| 说话方式 | 用词、语气、方言、节奏、表达风格    |

书中特别强调：人物的生理属性和自然属性不是装饰，而是故事素材库。人物的身高、家庭、学历、地域、经济能力，都可能变成后续情节中的独特难题。

---

## 8. 什么样的人物讨人喜欢

作者认为，读者通常会喜欢这些人物：

| 类型           | 原因         |
| ------------ | ---------- |
| 受压迫但抗争的人     | 激发同情和期待    |
| 坚持信仰的人       | 让人尊敬       |
| 勇敢、忠诚、有道德感的人 | 有正向吸引力     |
| 有弱点但复杂真实的人   | 更像真人       |
| 有牺牲精神的人      | 容易获得读者情感投入 |

但人物不能太单一。坏人也要有软肋，好人也可以有缺点。真实的人是复杂的，只属于一种标签的人物容易失真。

---

## 9. 人物展示方法

作者不建议直接写：

> 他很善良。
> 她很漂亮。
> 他武功很高。

更好的方法是让读者通过事件自己判断。

书中把人物介绍分成三种：

| 方法   | 说明                 | 效果        |
| ---- | ------------------ | --------- |
| 直接介绍 | 作者直接告诉读者           | 最省事，但容易无聊 |
| 他人介绍 | 通过别人评价、江湖传闻、外界反应展示 | 能制造期待     |
| 行动展现 | 通过人物做事展示性格         | 最有说服力     |

作者还提出“闻名—见面—相处”三阶段：先用传闻铺垫人物，再安排正式出场，最后通过长期行动展示人物魅力。

---

## 10. 对话写作方法

作者认为，对话不是闲聊，而是人物心理、情绪和目的的外化。对话和打架、吃饭、逃跑一样，都是人物行为。

对话至少要承担这些功能：

| 功能   | 说明             |
| ---- | -------------- |
| 推动情节 | 说完这段话，局面发生变化   |
| 塑造人物 | 通过用词、语气、节奏展示性格 |
| 制造冲突 | 对话双方目标不同       |
| 传递信息 | 但不能像作者说明书      |
| 制造悬念 | 话里有未说完的东西      |

对话的大忌是：人物明明都知道的信息，却故意说给读者听。作者强调，台词必须服务角色，而不是赤裸裸服务读者；信息要藏在人物“非说不可”的话里。

对话冲突的基本方法：

**让参与对话的人站在对立面。**

一方想知道秘密，另一方想保守秘密；一方想离开，另一方想挽留；一方想掩盖真相，另一方想逼问真相。这样对话就有张力。

---

## 11. 悬念设计方法

书中对悬念的定义很清楚：

**悬念就是让读者产生疑问，并且答案只能在你的故事里找到。**

如果读者没有疑问，就没有继续读下去的动力。作者甚至提出：如果小说前 100 字以内还不能让读者产生任何疑问，就要检查悬念是否设置失败。

悬念分为四层：

| 类型   | 作用              |
| ---- | --------------- |
| 超级悬念 | 贯穿全文，到结尾才解答     |
| 大悬念  | 贯穿一个阶段或大情节      |
| 场景悬念 | 在一个场景中提出并解决     |
| 段落悬念 | 每段末尾的小钩子，让读者继续读 |

作者强调，不能只靠一个大悬念撑完整本书。读者需要不断获得“小奖金”：小疑问、小答案、小反转，才能持续读下去。

---

## 12. 开头与切入点

切入点就是故事从哪里开始。作者反对“前三万字不吸引人，后面才好看”的写法，认为应该从最吸引人的地方开始讲。

切入点分三种：

| 切入方式 | 说明             | 适合情况         |
| ---- | -------------- | ------------ |
| 变化前  | 先展示主人公平静生活，再打破 | 需要让读者理解人物原状态 |
| 变化中  | 从意外正在发生时开始     | 适合强冲突、强戏剧感   |
| 变化后  | 意外已发生，从后果切入    | 适合制造悬念       |

所谓“变化”，就是主人公生活中的意外：离婚、战败、被追杀、家族被灭、公司破产、灾难爆发等。找到变化点，就能找到故事开头。

# 可直接套用的故事构思表

你以后构思任何小说/剧本，都可以先填这张表：

| 项目      | 填写                                         |
| ------- | ------------------------------------------ |
| 一句话故事   | 这是一个关于________的人，为了________，最终________的故事。 |
| 主人公是谁   | 身份：****；缺陷/挫折：****                         |
| 主人公要做什么 | 任务：____                                    |
| 为什么非做不可 | 动机：____                                    |
| 最大敌人/阻力 | 人/灾难/制度/自我缺陷：____                          |
| 结局      | 成功/失败/付出代价：____                            |
| 第一段变化点  | 主人公生活发生了什么意外：____                          |
| 第二段关卡 1 | 难题：****；解决方法：****；反转：____                  |
| 第二段关卡 2 | 难题：****；解决方法：****；反转：____                  |
| 第二段关卡 3 | 难题：****；解决方法：****；反转：____                  |
| 最终决战    | 主人公和敌人如何了结：____                            |
| 超级悬念    | 读者从开头到结尾最想知道的问题：____                       |
| 开头切入点   | 变化前 / 变化中 / 变化后                            |
| 主角讨喜点   | 读者为什么会在乎他：____                             |
| 主角说话方式  | 用词、语气、口头禅、节奏：____                          |

# 最精简版方法论

这本书可以压缩成一句话：

**先设计一个带着缺陷和目标的人，让他遇到越来越大的难题；再用人物小传保证他的行动合理，用对话和悬念让读者持续读下去，最后从最有吸引力的变化点切入。**
